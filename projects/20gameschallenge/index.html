20 Game Challenge

This challenge follows a set of games that start off simple and expand on each iteration with increased complexity. I am going to be doing this as a side hobby as I enjoy the process. I will be following the challenge for the most part but no promises that I do not get side tracked as I often do.


The progression will work from a simple game of Pong to much larger games but I plan to have fun adding new things to these older titles. I say fun for myself as I enjoy just making stuff up and seeing how it goes. There have been a few mock projects I have worked on for this. I plan to create a full experience from title screen to a full game loop and a possible end. They will not be as large as some of the original games but the base gameplay will be there.


I will be working with the Godot 4 game engine for the open-source and minimizing the walkthroughs of others making similar games. I will be looking up reference code for different aspects when necassary.
