Encapsulation/Decapsulation

As data moves down the OSI model different headers and trailers are added to the data to package it so the data is able to be sent and understood as it is processed on the other end. The headers and trailers act as packaging for the data to be sent correctly which is the Payload. As data is transferred, a set size limit is necessary to ensure information contained is sent properly. This limit on the size of the payload is the Maximum Transmission Unit (MTU).

Network Types and Characteristics

Peer-to-Peer
Peer-to-Peer relationships between systems work with two nodes connected directly to each other. These systems are able to simultaneously act as a client requesting information from the other and a server giving out services to another.

Client-Server
Client-Server relationships between systems create an established service provider who is the server and all other clients can request from the server.

Local Area Network (LAN)
Local Area Networks are a collection of nodes connected to a single network. These devices are able to communicate without having route data to another network. This makes most office networks and home networks.

Metropolitan Area Network (MAN)
Metropolitan Area Network cover the span of a city. This would be multiple different office building being connected together to create company network.

Wide Area Network (WAN)
Wide Area Network are networks that span over the world. These networks build up what is considered the Internet.

Wireless Local Area Network (WLAN)
Wireless Local Area Networks by communicating without a physical media to pass information. The data is transferred between nodes with wireless signals to each other. This is common in homes and offices that connect each device through wireless means.

Personal Area Network (PAN)
Personal Area Networks include the devices that are found on an individual person. These devices include smart watches, cell phones, wireless headphones, and other short range communication devices.

Campus Area Network (CAN)
Campus Area Networks are defined as covering multiple building within a small geographical region such as a college campus.

Storage Area Network (SAN)
Storage Area Networks are the network connections in a datacenter established to house storage for large amounts of data.

Service-related Entry Point

Demarcation Point
The demarcation point acts as the line between the service provider and the user. This is normally a box installation that brings internet connectivity to the home and where the connection to the users router or modem is. This also separates the task of repair as anything on the provider's side is up to them to fix but anything on the user's side is there problem to fix.

Smartjack
Smartjacks offer the ability for service providers to troubleshoot network issues without having to go all the way out to the site. It comes with a collection of utilities they can run to troubleshoot problems to the demarcation point.
